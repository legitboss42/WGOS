# Mission Router

The mission router maps user objectives to mission types in `config/mission-types.json`.

Supported mission types:

- WEBSITE_REBUILD
- WEBSITE_DESIGN
- CLIENT_WEBSITE_BUILD
- SEO_AUDIT
- SEO_CONTENT_REBUILD
- AEO_GEO_OPTIMIZATION
- ADSENSE_READINESS
- LEAD_GENERATION
- WEBSITE_AUDIT_FOR_LEADS
- OUTREACH_DRAFTING
- CRM_UPDATE
- ANALYTICS_REVIEW
- SEARCH_CONSOLE_REVIEW
- GA4_REVIEW
- CONTENT_PUBLISHING
- WEBSITE_SPEED_OPTIMIZATION
- CONTROLLED_BROWSER_TASK
- EMAIL_SETUP
- DEPLOYMENT_PREP
- DOCUMENTATION_UPDATE
- AGENT_SYSTEM_UPDATE
- GENERAL_STRATEGY
