# Workflow Certification

| Workflow | Status | Trigger | Departments |
| --- | --- | --- | --- |
| Workflow Orchestration | Certified | Any new mission that needs routing and dependency planning. | executive, operations, documentation |
| Website Rebuild Workflow | Certified | A website needs redesign, re-platforming, or major structural improvement. | executive, design, engineering, search, publishing, qa, documentation |
| SEO Growth Workflow | Certified | A site needs rankings, snippets, internal links, trust, or technical SEO improvements. | search, publishing, revenue, qa, documentation |
| Content Publishing Workflow | Certified | A content program or article batch needs planning, drafting, review, and publishing readiness. | publishing, search, qa, documentation |
| AdSense Readiness Workflow | Certified | A property needs policy-safe monetization readiness evaluation. | revenue, publishing, search, qa, documentation |
| Lead Research Workflow | Certified | The marketing team needs qualified leads and audit-ready outreach opportunities. | marketing, revenue, documentation |
| Human Approved Outreach Workflow | Certified | A campaign needs draft outreach and approval-gated follow-up plans. | marketing, client, documentation |
| Client Project Delivery Workflow | Certified | A client engagement needs end-to-end discovery through delivery preparation. | client, design, engineering, search, qa, documentation |
| Analytics Review Workflow | Certified | A business needs KPI review, reporting, and next-step recommendations. | revenue, marketing, executive, documentation |
| Weekly Agency Operations Workflow | Certified | The company needs a recurring operating review across delivery, growth, and risk. | executive, operations, marketing, revenue, documentation |
| Controlled Browser Workflow | Certified | A task needs browser observation with login, 2FA, or human-gated external steps. | executive, search, marketing, documentation |
| Documentation Update Workflow | Certified | A mission or task changed system state and requires durable records. | documentation, executive |
| QA And Release Workflow | Certified | A build or deliverable needs validation and release readiness review. | qa, engineering, documentation |

Certified workflows have named triggers, department ownership, documentation requirements, and Approval boundaries.
