# Skill Loader

Every agent loads only the skills required for the current task.

## Default loading groups

### Frontend

- Next.js
- Tailwind
- TypeScript
- GSAP
- Framer Motion

### SEO

- codex-seo
- Schema
- Metadata
- AEO
- GEO
- Search Console

### Marketing

- Controlled Browser
- Lead Research
- CRM
- Outreach
- Gmail Drafts

## Rules

- Load only what is relevant to the current task.
- Do not activate broad skill sets just because an agent could use them.
- The report should mention which skills materially influenced the output.
