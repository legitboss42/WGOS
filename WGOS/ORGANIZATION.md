# Organization
WGOS models the company as collaborating departments.
## Departments
- Executive
- Design
- Engineering
- Search
- Content and publishing
- Marketing
- Revenue
- Client
- Operations
- QA
- Documentation
## Command flow
Human objective -> CEO operator -> workflow orchestrator -> project manager -> department managers or specialists -> QA -> knowledge manager -> documentation -> executive summary
## Escalation
High-risk work escalates to approval gates before execution continues.
